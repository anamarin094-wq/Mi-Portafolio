Sensorial Store: Sistema de Inventario y Reporte de Ventas

Sistema de inventario de escritorio orientado a la gestión de productos cosméticos y perfumería, que implementa operaciones CRUD para el manejo de datos. Permite controlar existencias, precios y categorías, optimizando la organización de la información. El proyecto está diseñado con una interfaz intuitiva que facilita la interacción del usuario.

Sistema de Gestión de Personas (CRUD)

Aplicación de escritorio desarrollada para la gestión de registros de personas, que permite realizar operaciones CRUD sobre una base de datos. El sistema cuenta con una interfaz intuitiva que facilita la visualización y administración de la información, incluyendo búsqueda, actualización y listado completo de registros.

Sistema Web de Gestión de Productos y Categorías

El sistema facilita la organización de productos, asociándolos a sus respectivas categorías, e incluye funcionalidades como creación de registros, edición, visualización de detalles y eliminación. Cuenta con una interfaz clara, responsiva y orientada a mejorar la administración de datos.
